# Discovery Kids Channel

**Discovery Kids Channel** es una aplicación Android desarrollada en Kotlin que simula un canal de televisión retro con reproducción continua, programación por bloques, bumpers, comerciales, enseguidas y una interfaz visual inspirada en la experiencia de la TV clásica.

El proyecto está organizado por etapas de evolución: `alpha`, `beta` y `release`, donde cada versión representa un avance en el tiempo dentro de la línea del canal.

---

## 📺 Descripción

Este proyecto busca recrear la sensación de ver un canal infantil de televisión tradicional, con una programación lineal y elementos visuales y sonoros que acompañan la reproducción de contenido como si se tratara de una señal televisiva real.

Incluye:

- Reproducción automática de programas  
- Bumpers de canal  
- Comerciales dinámicos  
- Enseguidas de transición  
- Música de fondo durante programas  
- Pantalla completa inmersiva  
- Overlays visuales tipo CRT  
- Sistema de reanudación de sesión

## 🛠️ Instalación y compilación

### 📦 Requisitos

- Android IDE instalado  
- Java/Kotlin runtime  
- Permisos de almacenamiento  

### 📥 Instalación

1. Descarga el proyecto en `.zip`:

2. Extrae el archivo en tu dispositivo o PC.

### 📂 Importar

1. Abre **Android IDE**  
2. Selecciona **Import Project**  
3. Elige la carpeta del proyecto  
4. Espera sincronización de Gradle  

### ▶️ Ejecutar

1. Presiona **Run (▶)**  
2. Selecciona dispositivo o emulador  
3. La app se instalará automáticamente  

---

## ⚠️ Notas

- La primera sincronización puede tardar  
- Gradle descargará dependencias automáticamente  
- Dar permisos si el sistema lo solicita  
- Re-sincronizar si hay errores
